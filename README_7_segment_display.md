# Logisim-Works
A 7 segment display is implemented using logic gates for all inputs a,b,c,d,e,f,g.
Counter is clocked to run from 0 to 9 and the binary valued ouput is taken which serves as the inputs A,B,C,D.
A,B,C,D are the literals used to design the circuit for each pin on the display.
K-maps were used to get the equations for the circuit implementations.
